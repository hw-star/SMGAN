# SMGAN
PyTorch implementation of SMGAN: Remote Sensing Image super-resolution Reconstruction based on Cross-scale state Space Modeling

## Dependencies

- Python 3.9
- Python packages: `pip install numpy opencv-python lmdb pyyaml mamba`
