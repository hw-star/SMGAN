# SMGAN
PyTorch implementation of SMGAN: A Multi-Path Generative Adversarial Network for Remote Sensing Image Super-Resolution Reconstruction

## Dependencies

- Python 3.9
- Python packages: `pip install numpy opencv-python lmdb pyyaml mamba`
